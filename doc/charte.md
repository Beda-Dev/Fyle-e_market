# Charte Graphique — FYLE MARKET

## 1. Identité Visuelle

Le logo transmet une image :

- Moderne
- Chaleureuse
- Commerciale
- Accessible
- Dynamique

L’association du orange et du marron évoque :

- Le commerce
- L’énergie
- La confiance
- La proximité

Le design du site doit donc être :

- Épuré
- Moderne
- Très lisible
- Orienté expérience utilisateur
- Mobile-first
- Rapide et intuitif

# 4. Style UI / UX

## Direction Générale

Le site doit être :

- Minimaliste
- Moderne
- Fluide
- Élégant
- Commercial
- Responsive

---

## Espacement

- Beaucoup d’espace blanc
- Interface aérée
- Sections bien séparées

---

## Bordures

- Coins arrondis
- Radius recommandé : `12px à 20px`

---

## Ombres

- Légères
- Douces
- Modernes
